DirtyBoots

Web browser automation to obscure between genuine and fake browsing sessions

Automated Privacy Enhancement

DirtyBoots automates web browsing and mimics your browsing pattern, so your network traffic and online activity appear normal while away from keyboard / home.

Import your Firefox and/or Chrome browser histories and DirtyBoots will browse the internet in a similar manner. Use whitelists or blacklists to filter browsing history, or add a url-handler to execute when matched urls match specified regex.

  • Browser Support

    Firefox, Chrome, and Internet Explorer

  • Flexibility

    .conf files and CLI args provide easy scripting

  • Features

    White/Black lists, manual searches, error-prone typing, and more

PrivatePanda

Hardens Firefox's security and privacy settings.

A More Secure and Private Browsing Configuration

PrivatePanda is a script for Firefox to harden the security and privacy settings of the browser. PrivatePanda changes settings in the about:config to remove geolocation, social media, Google services and servers. Additionally, PrivatePanda cleans up all browsing history and cookies on every shutdown.

This script is easy to install (just ctrl+c & ctrl+v into your profile directory) and easily removed. Feature versions will be including a configuration wizard / walk-through.

  • Blocks

    Google, social media, geo-location, click-tracking, and more!

  • Changes

    DuckDuckGo for searches, disables autoplay, displays unicode URLs.

  • Cleans Up

    Clears all history and cache files on shutdown.

KnowledgeBomb

A compilation of free and open-source tools for an offline encyclopedia, library, and navigation workstation.

[ Launching Soon ]

Live-USB Preloaded for Privacy and Education

KnowledgeBomb is a content-focused project to provide a mobile, offline internet resource-center. All data sources are either in the public domain or licensed under Creative Commons.

Wikipedia, global navigation, public domain books, and open-source software on a Live-USB offers common internet use-cases on an air-gapped computer.

The Live-USB comes in several OS flavors to suite individual needs. If you'd prefer just to maintain a copy of the data sources yourself, I've included them in the Resources section below.

  • Use Cases

    Offline reading, international travel, remote locations, time-capsule backup, and more

Resources

Free and open-source tools for offline reading and research.

Below is a collection of free resources and tools that I've found quite useful to preload onto my personal systems/laptop. Many of these resources are part of the KnowledgeBomb project. This is a quick way to have a knowledge-center with core internet use-cases covered (Wikipedia, maps / navigation, research, ebooks/reading). These are great resources to have loaded onto air-gapped systems or when venturing into an area with minimal internet access.

My primary goal with this collection was to ensure a fully offline solution with free open-source software and public domain and Creative Commons data sources. When travelling, I backup my laptop, reimage it with a factory-new install, and fill it up with the resources below. This helps avoid carrying a system with private/personal information, which is especially useful when travelling internationally.

Library

For offline reading, I use an offline dump of Wikipedia, the public domain ebook collection, Project Gutenberg, and manage the books with Calibre. This is a simple, free, and open-source stack for offline reading and research - perfect for travel, Internet-less locations, and truly private offline browsing.
